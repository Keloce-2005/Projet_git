# Code of Conduct

## Objectif

Ce projet est un travail collaboratif.
Chaque membre doit respecter les autres et suivre les règles du projet.

## Règles

- Respecter les autres membres
- Ne pas modifier le travail des autres sans accord
- Utiliser les issues pour discuter
- Utiliser les pull requests pour les modifications
- Respecter le workflow Git défini

## Collaboration

Toutes les contributions doivent passer par :

- une issue
- une branche
- une pull request

## Comportement interdit

- push direct sur main
- suppression de branches sans accord
- commits non signés
- modifications sans issue

## Respect

Le respect et la communication sont obligatoires.
